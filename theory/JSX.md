## 확장된 JavaScript 문법
### JSX

- `{}`안에 데이터타입(배열, 문자열) 삽입가능, 연산자 삽입 가능
- JavaScript 안에 HTML 문법 쓸 수 있음
- `{}`내부에는 JavaScript 표현식만 넣을 수 있다.
    - if, for문 불가
- JSX에서 숫자, 문자열 또는 배열의 값만 렌더링 된다.
- boolean, undefine, null 값, 객체는 셋팅 안됨
    - 객체.점표기법은 사용가능
- 모든 태그는 닫혀있어야 한다.
- 최상위 태그는 반드시 하나여야만 한다.<main>><div> 등
